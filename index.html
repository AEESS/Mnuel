Introduction au HTML

Le HTML (HyperText Markup Language) est le langage de balisage standard utilisé pour créer des pages web. Il décrit la structure d'une page web et consiste en une série d'éléments qui indiquent au navigateur comment afficher le contenu.

Les éléments HTML sont les blocs de construction des pages web. Ils sont représentés par des balises qui entourent le contenu pour indiquer comment il doit être affiché.

<!DOCTYPE html>
<html>
<head>
    <title>Ma première page</title>
</head>
<body>
    <h1>Bonjour le monde !</h1>
    <p>Ceci est mon premier paragraphe.</p>
</body>
</html>

Dans cet exemple, nous pouvons voir :

  • <!DOCTYPE html> - Déclare le type de document
  • <html> - L'élément racine de la page
  • <head> - Contient les métadonnées
  • <title> - Définit le titre de la page
  • <body> - Contient le contenu visible
  • <h1> - Un titre de niveau 1
  • <p> - Un paragraphe

Exercice pratique

Créez une page HTML simple avec un titre et deux paragraphes. Le titre doit dire "Bienvenue sur mon site" et les paragraphes doivent contenir quelques phrases de votre choix.
